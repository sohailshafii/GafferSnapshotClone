# GafferSnapshotClone

Reimplementation of Gaffer's Snapshot Compression [technique](https://gafferongames.com/post/snapshot_compression/). The original code can be found [here](https://gist.github.com/gafferongames/bb7e593ba1b05da35ab6).

It is recommended to create a VS or XCode project first via `cmake`, then build and run inside of that IDE. For XCode, you would use `cmake -G Xcode`.
